# 最小路径和

题目

```bash
给定一个包含非负整数的 m x n 网格，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

说明：每次只能向下或者向右移动一步。

示例:

输入:
[
  [1,3,1],
  [1,5,1],
  [4,2,1]
]
输出: 7
解释: 因为路径 1→3→1→1→1 的总和最小。
```

题解

这是一道很经典的需要用动态规划求解的问题，动态规划状态直接定义为第m行n列的最小总和，可以用上一步的值求出，很容易的一个动态规划。代码如下。

```C++
class Solution {
public:
    int minPathSum(vector<vector<int>>& grid) {
        vector<vector<int>> dp = grid;
        int m(dp.size()), n(dp[0].size());
        if(m==0||n==0) return 0;
        for(int i=1; i<m; i++) dp[i][0] += dp[i-1][0];
        for(int i=1; i<n; i++) dp[0][i] += dp[0][i-1];
        for(int i=1; i<m; i++)
        {
            for(int j=1; j<n; j++)
            {
                dp[i][j] += min(dp[i-1][j], dp[i][j-1]);
            }
        }
        return dp[m-1][n-1];
    }
};
```
