# 算法题目总结

主要讲题目进行以下类别的分类，方便后续进行查看。

**大概思路**：二分，快排，归并，求众数，dp，bfs，dfs，数组合并，链表倒序，是否有环，2链表是否相交，链表排序，最长子序列，最长回文子串，背包等dp问题，二叉树的各种遍历，递归非递归，序列化与反序列化，求深度，判断平衡树，最近公共祖先，topk，第k大，全排列等。
